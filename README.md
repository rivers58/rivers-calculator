# Rivers Calculator / R-CALC

手機版計算機 PWA。版面參考使用者提供的計算機 App：上方完整計算紀錄、日期分段，下方大按鍵，支援括號四則運算。

## 安裝與啟動

```bash
npm install
npm run dev
```

打開：

```text
http://localhost:3000
```

## 部署

建議推到 GitHub 後用 Vercel 部署。iPhone 使用 Safari 打開網址後，點分享，選「加入主畫面」。

## V1 功能

- 加、減、乘、除
- 小數點
- 括號運算
- 退格
- 清除
- 計算紀錄
- 日期分段
- localStorage 保留紀錄
- PWA manifest
